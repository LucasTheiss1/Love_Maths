# Love Maths – Essential JavaScript Project
## 📚 Project Overview

Love Maths is a hands-on JavaScript project developed as part of my Web Development Level 5 course.

The purpose of this project is to practice core JavaScript fundamentals while building an interactive math quiz game that runs directly in the browser.

The application generates random math questions and allows the user to test their arithmetic skills while tracking their score.
